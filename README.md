
# 📊 Financial Planning & Forecasting Dashboard

A Business Analysis project demonstrating financial planning, budgeting, variance tracking and forecasting using SQL and Excel frameworks.

## 🎯 Project Objective

- Track actuals vs budget across departments
- Monitor variance trends month-over-month
- Forecast quarterly and annual spend
- Support decision-making through KPI dashboards

## 📁 Files in this Repo

| File | Description |
|------|-------------|
| BRD_Financial_Planning.md | Business Requirements Document |
| Dashboard_Wireframe.md | Dashboard layout and KPI definitions |
| queries.sql | SQL queries for variance, trend and forecast |
| sample_financial_data.csv | Sample dataset |

## 📋 Business Requirements Summary

| ID | Requirement | Priority |
|----|-------------|----------|
| BR-01 | Display actual vs budgeted spend by department | High |
| BR-02 | Calculate variance in absolute and percentage terms | High |
| BR-03 | Support monthly and quarterly aggregation | High |
| BR-04 | Enable drill-down by cost center | Medium |
| BR-05 | Provide rolling 3-month forecast | Medium |
| BR-06 | Flag departments exceeding 10% negative variance | High |

SQL | Excel | Business Process Modeling | Stakeholder Analysis
